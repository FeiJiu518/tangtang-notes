# 🍬 糖糖便签

一款精致的桌面便签应用，帮助你管理各类信息。

![糖糖便签](./screenshots/preview.png)

## ✨ 功能特性

- 📝 **多种便签类型** - 支持文本、密码、链接、日期、消费记录 5 种类型
- 🔐 **隐私保护** - 私密便签需要 PIN 码验证，密码可设为私密
- 👥 **多用户支持** - 不同用户数据完全隔离
- 🔑 **安全机制** - 支持安全问题找回密码
- 🎨 **自定义分类** - 可自定义分类图标和颜色
- 📌 **侧边吸附** - 拖到屏幕边缘自动收缩为迷你模式，置顶显示
- 🖼️ **图片支持** - 便签支持添加图片
- 👤 **个性头像** - 支持上传并裁剪自定义头像
- 🎯 **圆角窗口** - 现代化无边框圆角设计

## ✨ 功能特点

### 📝 多种便签类型
- **账号密码** - 安全存储各类账号信息
- **网址收藏** - 收藏常用网站链接
- **重要日期** - 记录重要日子并设置提醒
- **报销记录** - 追踪报销状态和金额
- **会员管理** - 管理各种会员订阅，自动计算到期时间
- **通用备忘** - 自由记录任何文本内容

### 🔐 安全可靠
- 用户密码 SHA-256 哈希加密存储
- 支持 PIN 码保护私密便签
- 本地存储，数据不上传云端

### 🎨 简洁易用
- 清爽的界面设计
- 支持自定义分类图标和颜色
- 分类拖拽排序
- 边缘吸附收缩，不占用桌面空间

## 🖥️ 安装使用

### 下载安装
1. 前往 [Releases](../../releases) 页面
2. 下载最新版本的 `糖糖便签.Setup.x.x.x.exe`
3. 双击安装，按提示完成安装

### 从源码运行
```bash
# 克隆仓库
git clone https://github.com/你的用户名/tangtang-notes.git

# 进入项目目录
cd tangtang-notes/info-notes-app

# 安装依赖（只需一次）
npm install

# 开发模式运行
npm run electron:dev

# 编译打包
npm run electron:build:win
```

## 📖 使用说明

### 首次使用
1. 打开应用，点击「注册」创建账号
2. 设置用户名、密码和安全问题
3. 登录后即可开始使用

### 创建便签
1. 点击左上角「+ 新建便签」
2. 选择便签类型和分类
3. 填写内容并保存

### 窗口吸附
- 将窗口拖到屏幕左/右边缘，自动收缩为悬浮条
- 鼠标悬停在悬浮条上，窗口自动展开
- 可拖拽悬浮条切换左右位置

## 📋 更新日志

### v1.1.0 (2025-01-06)

#### 🔐 安全性增强
- 用户密码使用 SHA-256 哈希加密存储
- 安全问题答案加密存储
- "记住密码"功能采用对称加密（XOR + Base64）

#### 💎 新增会员管理功能
- 新增「会员管理」类别（紫色钻石图标）
- 支持多种会员类型：连续包月/年、月卡、季卡、年卡、自定义
- 自动根据会员类型计算到期日期
- 会员状态自动判断：🟢生效中 / 🟡即将到期 / ⚫已过期
- 会员统计：累计花费、本月支出、年度支出
- 到期提醒 + 自动续费关闭提醒

#### 🚪 单窗口登录限制
- 同一账号只能在一个窗口登录，防止数据冲突
- 基于文件锁机制，支持多进程检测

#### 📁 分类管理优化
- 支持拖拽调整分类顺序
- 删除分类前弹出确认对话框，防止误删
- 创建分类时根据类型自动填充默认图标和颜色

#### 📎 附件功能增强
- 支持拖拽上传图片
- 支持上传文件：PDF、Word（.doc/.docx）、TXT
- 附件可预览、下载、删除

#### ⏰ 提醒功能优化
- 提醒时间精确到秒（每秒检测一次）
- 提醒触发后自动关闭，避免重复提醒
- 过期时间自动验证，不能设置过去的提醒
- 卡片上直接显示提醒时间

#### 🔀 排序功能
- 报销记录：按创建时间 / 按购买日期
- 重要日期：按创建时间 / 按提醒日期
- 会员管理：按创建时间 / 按到期日期 / 按累计花费
- 支持升序 ↑ / 降序 ↓ 切换

#### 🔍 筛选功能
- 报销记录：按状态筛选（未开发票/已申请未出票/已开发票未报销/已报销）
- 重要日期：按过期状态筛选（未过期/已过期）
- 会员管理：按状态筛选（生效中/即将到期/已过期）
- 支持多选筛选，选中所有子项自动变为"全部"

#### 📋 右键菜单
- 编辑框：全选、剪切、复制、粘贴
- 非编辑框选中文字：复制

#### 🖥️ 窗口功能
- 侧边栏吸附支持左右切换（拖到屏幕另一侧自动切换）

#### 👤 账号功能优化
- 登录过的账号自动记住
- 可选"记住密码"，下次自动填充
- 已保存账号显示用户头像
- 登录界面内容过多时可滚动

#### 📝 报销记录优化
- 新增状态：「已申请未出票」
- 状态顺序调整：未开票 → 已申请未出票 → 已开发票未报销 → 已报销

#### ✨ 其他优化
- 新建便签时默认选中当前分类
- 文本选择蓝色高亮样式

## 📝 便签类型说明

| 类型 | 说明 | 特殊功能 |
|-----|------|---------|
| 文本 | 普通文本便签 | 支持图片 |

## 🔒 安全说明

- 所有数据存储在本地，不会上传到任何服务器
- 密码类便签支持 PIN 码二次保护
- 支持账号密码 + 安全问题双重验证
- 数据存储位置: `C:\Users\<用户名>\AppData\Roaming\tangtang-notes\`
